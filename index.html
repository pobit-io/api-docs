Pobit API Documentation

포빗 플랫폼의 모든 API 서비스에 대한 종합적인 문서화 허브입니다. 각 서비스별 상세한 엔드포인트 정보와 사용법을 확인하세요.

Pobit Account API

사용자 인증, 계정 관리, 프로필 정보를 처리하는 핵심 API 서비스입니다. JWT 기반 인증과 소셜 로그인을 지원합니다.

  • JWT 토큰 기반 인증
  • Google 소셜 로그인
  • 사용자 프로필 관리
  • 역할 기반 접근 제어
  • 이메일 마스킹 처리

Pobit Client API

클라이언트 애플리케이션 관리, 설정, 배포를 담당하는 API 서비스입니다. 다양한 클라이언트 환경을 지원합니다.

  • 클라이언트 애플리케이션 관리
  • 설정 및 환경 변수 관리
  • 배포 및 버전 관리
  • 실시간 모니터링
  • 성능 최적화
