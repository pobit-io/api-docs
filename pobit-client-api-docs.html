💹 Pobit Client API (1.0.0)

Download OpenAPI specification:

Pobit Development Team: support@pobit.io URL: https://pobit.io License: MIT

Pobit 클라이언트 거래소 데이터 API 문서

🔄 실시간 데이터 특징

  • USD/KRW 환율: Google Finance 실시간 데이터
  • 스테이블코인 가격: 업비트, 빗썸, 코인원 실시간 API
  • Rate Limiting: 거래소별 API 제한 자동 준수
  • 장애 복구: 일부 거래소 실패 시에도 가능한 데이터 제공

📊 주요 기능

  • 통합 환율 데이터: USD/KRW 환율 + 스테이블코인 가격 통합 조회
  • 스테이블코인 추적: USDT, USDC 실시간 가격 모니터링
  • 거래소별 비교: 업비트, 빗썸, 코인원 가격 차이 분석
  • 가격 변동률: 전일 종가 대비 변화율 계산

🎯 지원 거래소

  • 업비트 (UPBIT): 10 requests/1초
  • 빗썸 (BITHUMB): 150 requests/1초
  • 코인원 (COINONE): 1,200 requests/60초

📈 데이터 정확성

  • 실시간 API: 거래소 공식 API 직접 호출
  • UTC 기준: 전일 종가는 UTC 0시 기준으로 통일
  • 오류 처리: 거래소별 독립적 오류 처리로 안정성 확보

💡 API 사용 가이드

  1. 단순 환율 조회: /exchange/usd-to-krw
  2. 스테이블코인 가격: /exchange/stablecoin-prices
  3. 통합 데이터: /exchange/data (권장)

💹 Exchange Rate (환율)

실시간 환율 조회

USD/KRW 환율 + 스테이블 코인 가격 통합 조회

USD to KRW 환율과 함께 주요 스테이블 코인(USDT, USDC)의 가격 정보를 통합하여 제공합니다.

제공 정보:

  • USD/KRW 환율 (Google Finance)
  • 스테이블 코인 가격 정보 (업비트, 빗썸, 코인원)
  • 현재 가격 및 전일 종가 (UTC 0시 기준)
  • 가격 변화량 및 변화율

Rate Limiting:

  • 거래소별 API 제한 준수
  • 일부 거래소 실패 시에도 다른 데이터 제공

데이터 신뢰성:

  • 실시간 API 데이터
  • 거래소별 독립적 조회

Responses

개발 서버

http://localhost:3001/exchange-rate

운영 서버
